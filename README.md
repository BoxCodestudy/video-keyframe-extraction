# 基于深度特征聚类的视频关键帧提取与事件检测系统

面向监控视频的智能浓缩与异常筛查系统，提供从原始视频输入到精简摘要视频输出的全流程自动化处理。

## 主要功能

1. **双源视频输入** — 支持从本地文件选择视频（MP4/AVI/MKV/MOV/FLV/WMV）或通过摄像头实时录制（自动保存为 MP4），录制窗口提供实时画面预览和录制时长指示
2. **多格式视频自适应帧采样与分解** — 按可配置采样步长将视频分解为 JPEG 帧序列
3. **入侵事件检测与时间轴标记** — 基于 MOG2 背景建模 + HOG 行人检测，聚合连续入侵帧为结构化事件
4. **关键帧智能筛选** — 使用 ResNet-18 预训练模型提取 512 维深度特征，KMeans 聚类去重并推荐代表性关键帧
5. **视频合成与导出** — 基于 FFmpeg 多片段合成，结构化目录导出摘要视频和独立事件片段，支持自动导出

│   ├── detection.py               # 入侵事件检测

## 技术栈

- **GUI：** Tkinter（现代简洁风 UI）
- **视频处理：** OpenCV（MOG2 背景建模、HOG 行人检测、摄像头采集）
- **深度学习：** PyTorch + torchvision（ResNet-18 预训练模型特征提取）
- **聚类分析：** Scikit-learn（KMeans / DBSCAN）
- **视频合成：** imageio-ffmpeg（FFmpeg trim+concat），OpenCV 降级备选
- **图像处理：** Pillow

## 环境要求

- Python 3.8+
- PyTorch ≥ 2.0.0
- torchvision ≥ 0.15.0
- OpenCV ≥ 4.5.0
- Scikit-learn ≥ 1.0.0
- NumPy ≥ 1.21.0
- Pillow ≥ 9.0.0
- imageio-ffmpeg ≥ 0.4.0（推荐用于视频合成）

### 安装依赖

```bash
pip install -r requirements.txt
```

无 ML 环境下系统自动降级为均匀采样策略，仍可使用基本功能。

## 快速开始

### 启动应用

```bash
python -m src.gui_app
```

### 操作流程

**方式一：从文件选择**
1. 点击顶部 **"📁 视频来源 ▾"** → **"📂 从文件选择"**
2. 选择待处理的视频文件，预览区显示视频信息
3. 点击 **"▶ 开始处理"** → 4 步流水线自动执行，底部进度条实时显示进度
4. 处理完成后自动导出到输出目录（可在设置中关闭自动导出）

**方式二：摄像头录制**
1. 点击顶部 **"📁 视频来源 ▾"** → **"🎥 从摄像头录制"**
2. 录制窗口弹出，等待摄像头就绪后点击 **"🔴 开始录制"**
3. 点击 **"⏹ 停止录制"**，视频自动加载到主界面
4. 点击 **"▶ 开始处理"** 执行流水线

## 配置说明

点击 **"⚙ 设置"** 可自定义：
- **输出文件默认路径** — 处理结果的保存位置（默认 `output/`）
- **临时文件路径** — 视频分解时的帧缓存位置（默认 `temp/`）
- **摄像头录制保存路径** — 录制视频的保存位置（默认 `saved_video/`）
- **处理参数** — 采样帧间隔、入侵检测阈值、运动/行人权重等
- **界面偏好** — 自动导出、自动打开输出文件夹、记住上次路径

配置自动保存在 `config/video_processor_config.json`。
