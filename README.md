# 基于深度特征聚类的视频关键帧提取与事件检测系统

面向监控视频的智能浓缩与异常筛查系统，提供从原始视频输入到精简摘要视频输出的全流程自动化处理。主要功能包括：多格式视频的自适应帧采样与分解；基于背景建模（MOG2）与梯度直方图（HOG）的入侵事件检测与时间轴标记；基于 ResNet-18 深度特征与 KMeans 无监督聚类的关键帧去重与代表性锚点推荐；以及基于 FFMPEG 的多片段合成导出。系统提供图形化操作界面，支持一键执行完整流水线，并允许在无深度学习环境下降级运行基础均匀采样功能。

## 项目结构

```
├── config/              # 配置文件目录
│   └── video_processor_config.json
├── data/                # 输入数据目录（存放待处理的视频）
├── logs/                # 运行日志目录
├── output/              # 输出结果目录
├── src/                 # 源代码目录
│   └── main.py          # 主程序
├── temp/                # 临时文件目录
│   └── frames/          # 视频分解后的帧缓存
└── README.md            # 本文件
```

## 技术栈

系统采用 Python 生态构建，图形界面基于 Tkinter 实现。视频处理管线以 OpenCV 为核心，使用 MOG2 自适应高斯混合模型进行背景建模与运动前景提取，结合 HOG 描述子实现行人目标检测，通过时序连续性分析将逐帧检测结果聚合为结构化事件。关键帧筛选模块使用 PyTorch 加载预训练 ResNet-18 模型提取深度特征向量，采用 Scikit-learn KMeans 聚类算法对视觉相似帧进行分组，并在各簇内选取距质心最近帧作为输出锚点。视频合成与导出调用 FFMPEG 实现高效编解码。系统采用模块化分层设计，将 GUI 层、业务逻辑层与算法层解耦，检测与特征提取组件可在无 GPU 环境下自动降级。

## 环境要求

- Python 3.8+
- PyTorch (支持 CUDA)
- OpenCV
- Scikit-learn
- PIL/Pillow
- imageio-ffmpeg (推荐用于视频合成)

### 安装依赖

```bash
pip install opencv-python numpy torch torchvision scikit-learn pillow
```

如果没有 ML 环境，代码仍可运行基础功能（均匀采样关键帧）。

## 使用方法

### 1. 运行程序

```bash
cd src
python main.py
```

### 2. 操作流程

1. 点击 **"选择视频"**，选择待处理的视频文件
2. 点击 **"运行完整流水线"**（或勾选自动流水线）
3. 程序将依次执行：
   - 视频分解为帧
   - 入侵事件检测
   - 关键帧识别
   - 等待用户确认后生成导出
4. 点击 **"生成视频"**，选择输出目录

## 配置说明

配置会自动保存在 `config/video_processor_config.json` 中。
